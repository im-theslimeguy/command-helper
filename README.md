Explore Minecraft commands quickly and easily! Command Helper displays concise command information right in your HUD, helping you understand parameters and usage without reading long documentation.

## Features

Hold **Alt** (or any key you set in Keybinds) to show a tooltip at your cursor, which displays information about the command you're trying to run. Info includes what the command will do and its parameters.

### Ask the assistant

In the help window, you'll find an **_Ask Assistant_** button. By entering your own Gemini API key, you can chat directly with the AI in-game to get help with commands and parameters. 
> *(Note: To use this feature, you must acknowledge the geolocational availability and agree to Google's Privacy Policy and Terms of Service).*

### Important Disclaimer

The AI features are powered by Google's Gemini 2.0 Flash model. To comply with the Minecraft EULA and standard platform guidelines, **you must provide your own API key** – we cannot bundle a shared or developer API key with the mod. 

**Privacy & Data Security:** Your API key is stored safely on your local machine and is *only* used to communicate with Google's servers. When you use the assistant, only your written prompt, your current Minecraft version, and your local time/date are sent to Google to provide accurate context. Absolutely no data is collected by the mod creators, and nothing is shared with any other third parties.

### Don't know how to get an API key?

Check the mod's website for more information.
