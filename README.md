Explore Minecraft commands quickly and easily! Command Helper displays concise command information right in your HUD, helping you understand parameters and usage without reading long documentation.

## Features in this version

Hold **Alt** (or any key you set in Controls) to show a tooltip at your cursor, which displays information about the command you're trying to run. Info includes what the command will do and its parameters.

## Collaboration

Other developers can use Command Helper to display help information for their custom commands too. Visit the GitHub site for more information.
